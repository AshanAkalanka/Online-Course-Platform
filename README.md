🎓 EduFlow – Online Course Platform

EduFlow is a full-stack online course platform. It helps students easily browse courses, enroll, and watch lessons online. This project includes both **student-side** and **admin-side** features with a modern and user-friendly design.

Built using **React, Node.js, Express, and MySQL**, EduFlow provides a smooth learning experience with secure authentication and course management features.


✨ Features

👨‍🎓 Student Side

* Register and login
* Browse available courses
* Search and filter courses
* View course details
* Enroll in courses
* Watch lessons
* Mark lessons as completed
* Track learning progress and more...

🛠️ Admin Side

* Admin login
* Admin dashboard
* Add, edit, and delete courses
* Upload course thumbnail images
* Add, edit, and delete lessons
* View all users
* Change user roles
* Delete users and more...

🔐 Security Features

* JWT authentication
* Password hashing with bcrypt
* Role-based access control
* Protected routes
* Input validation
* Secure API handling
* Rate limiting
* Helmet middleware


#💻 Tech Stack

🚀 Purpose of the Project

The main purpose of this project is to create a simple and efficient online learning platform where students can learn easily and admins can manage course content smoothly.
It is a full-stack project for learning how to build a real-world education platform with both frontend and backend features.
